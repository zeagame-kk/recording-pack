# Changelog

บันทึกการเปลี่ยนแปลงของระบบ Packing Recorder Lite

## [v0.5.1] - 2026-06-21
### Fixed
- **Shared Drive Support**: เพิ่มพารามิเตอร์ `corpora=allDrives`, `includeItemsFromAllDrives=true` และ `supportsAllDrives=true` ในการเชื่อมต่อ Google Drive API ทั้งใน `index.html` และ `downloader.html` เพื่อแก้ปัญหาการค้นหาวิดีโอไม่พบบน Shared Drive ของอีเมลบัญชีอื่น (เช่น `natnicha.rs1944@gmail.com`)
- **Cache Busting**: อัปเดตเวอร์ชันบนหน้าเว็บเป็น `v0.5.1` เพื่อป้องกันเบราว์เซอร์ (โดยเฉพาะ Safari บนมือถือ) จำไฟล์เวอร์ชันเก่า

## [v0.5.0] - 2026-06-21
### Added
- เพิ่มฟังก์ชัน **Auto Cleanup History** ลบประวัติการแพ็คสินค้าเก่าที่เกิน 3 วันออกอัตโนมัติเมื่อเปิดหน้าเว็บ (เพื่อป้องกันปัญหาข้อมูลรกและบราวเซอร์หนักเกินไป)
- เพิ่มฟังก์ชัน **Daily Count** แสดงจำนวนออเดอร์ที่แพ็คและอัปโหลดสำเร็จในวันนี้ของเครื่องนั้นๆ ให้พนักงานเห็นชัดเจน (ตรงหัวข้อ History)
- **New 2-Column Layout (Downloader)**: ปรับปรุงหน้า `downloader.html` ให้เป็นแบบ 2 ฝั่ง (ซ้าย: ค้นหาด้วยออเดอร์/ดูวิดีโอ, ขวา: ค้นหาด้วยวันที่/ดูกราฟ)
- **Line Graph Analytics (Downloader)**: เพิ่มกราฟเส้นรายชั่วโมง (`Chart.js`) สำหรับดูสถิติการแพ็คสินค้ารวมจากทุกเครื่อง
- ล็อกตำแหน่ง **Footer** ให้ตรึงอยู่ด้านล่างสุดของหน้าจอเสมอ เพื่อไม่ให้กวนสายตาขณะดึงวิดีโอ

## [v0.4.0] - 2026-06-21
### Removed
- ยกเลิกระบบแปลงไฟล์ FFmpeg เป็น MP4 อีกครั้ง (Rollback กลับไปเป็น v0.2.0) เนื่องจากปัญหาการทำงานบนเบราว์เซอร์เครื่องผู้ใช้ที่ไม่เสถียร 
- ระบบกลับมาใช้งานการเปิดไฟล์ `.webm` ด้วย Video Player บนเบราว์เซอร์แทน เพื่อเน้นความรวดเร็วและความเสถียรสูงสุดในการทำงานของพนักงาน

## [v0.3.0] - 2026-06-21
### Added
- นำระบบ FFmpeg (v0.12 Single-Threaded) กลับมาใส่ในหน้า Downloader อีกครั้ง เพื่อแปลงไฟล์ WebM เป็น MP4 สำหรับการนำไปส่งผ่านแอปพลิเคชัน LINE บน iOS

## [v0.2.0] - 2026-06-21
### Added
- เพิ่มระบบ **Version Control** ใส่หมายเลขเวอร์ชันให้ระบบและทำระบบ Changelog
- เปลี่ยนรูปแบบหน้า **Downloader** ใหม่ให้ทำงานเบาและรวดเร็วขึ้น
- **Video Player**: เพิ่มระบบเล่นวิดีโอ (WebM) โดยตรงบนหน้าเว็บ (เล่นได้ทันทีหลังดาวน์โหลดเสร็จโดยไม่ต้องพึ่งโปรแกรมภายนอก)

### Removed
- นำระบบแปลงไฟล์ FFmpeg (WebM เป็น MP4) ออกจากเบราว์เซอร์ เพื่อลดปัญหา Cache ของ Service Worker และแก้ปัญหาเครื่องผู้ใช้งานกระตุก/ทำงานล้มเหลว (Failed to fetch)

## [v0.1.0] - 2026-06-21
### Added
- สร้างหน้า **Downloader (downloader.html)** สำหรับผู้จัดการใช้ค้นหาและดาวน์โหลดวิดีโอจาก Google Drive โดยอ้างอิงจาก Order Number
- เริ่มต้นใช้ **FFmpeg (WebAssembly)** ในการลองแปลง WebM เป็น MP4 บนฝั่ง Client
- เปลี่ยนมารวมโค้ดจากหลายไฟล์เป็นไฟล์เดียว (Single File Component) เพื่อให้ง่ายต่อการ Deploy ขึ้น GitHub Pages

### Changed
- ตั้งค่าการสแกน Barcode ให้ทำงานทันทีเมื่อกดปุ่ม Enter (โดยไม่ต้องกดปุ่ม Start Recording เอง) กล้องจะเริ่มบันทึกและโชว์หน้าจอเมื่อสแกนออเดอร์เท่านั้น
- ย้ายระบบการ Deploy ไปที่ **GitHub Pages** เป็นหลัก เพื่อให้พนักงานใช้งานง่ายขึ้น

## [v0.0.1] - 2026-06-21 (Phase 0 Initial)
### Added
- สร้างโปรเจกต์ **Packing Recorder Lite** เริ่มต้นสำหรับการบันทึกการแพ็คสินค้า
- พัฒนาระบบบันทึกวิดีโอจากกล้อง Webcam (MediaRecorder) พร้อมฝังหมายเลข Order ลงในวิดีโอ (Canvas Overlay)
- เชื่อมต่อ API ของ Google Drive เพื่ออัปโหลดไฟล์วิดีโอขึ้น Google Shared Drive อัตโนมัติ (บันทึกเป็นไฟล์ `.webm`)
- เพิ่มระบบประวัติการแพ็คสินค้า (IndexedDB) เพื่อเก็บข้อมูลว่าเคยแพ็คอะไรไปแล้วบ้างในเครื่องของพนักงาน
