# 📦 คู่มือติดตั้งและใช้งาน Packing Recorder (Chrome Extension สำหรับ BigSeller)

ส่วนเสริม (Chrome Extension) นี้ช่วยฝังกล้องเว็บแคมและระบบบันทึกวิดีโอแพ็คสินค้าเข้าไปในหน้าจอ **BigSeller - สแกนและตรวจสอบ** (Scan to Inspection) โดยดึงข้อมูลเลขแทร็คกิ้งและเริ่มบันทึก/อัปโหลดไฟล์ขึ้น Google Drive อัตโนมัติ

---

## 🚀 วิธีติดตั้งส่วนเสริมบน Google Chrome

1. เปิดเบราว์เซอร์ **Google Chrome**
2. พิมพ์ลิงก์นี้ในช่องที่อยู่เพื่อไปยังหน้าส่วนเสริม: `chrome://extensions/`
3. เปิดใช้งาน **"โหมดนักพัฒนาซอฟต์แวร์" (Developer mode)** ที่มุมบนขวาของหน้าต่าง
4. คลิกปุ่ม **"โหลดแอปที่คลายซิปแล้ว" (Load unpacked)** ที่มุมซ้ายบน
5. เลือกโฟลเดอร์ `extension` ที่อยู่ในโปรเจกต์นี้:
   `[พาธของโปรเจกต์]/recording_pack/extension`
6. ปักหมุดไอคอนส่วนเสริม **Packing Recorder** บนแถบเครื่องมือ Chrome เพื่อความสะดวกในการใช้งาน

---

## ⚙️ ขั้นตอนการตั้งค่า Google Drive (ทำเพียงครั้งแรก)

1. คลิกที่ไอคอนส่วนเสริม **Packing Recorder** (รูปกล่อง 📦) บนแถบเครื่องมือ Chrome
2. คลิกปุ่มฟันเฟือง **⚙️** เพื่อเปิดการตั้งค่า
3. กรอก **Google Client ID** และ **Google Folder ID** (ค่าเริ่มต้นมีใส่ไว้ให้แล้วเป็น Shared Drive เดิม)
4. คลิก **💾 Save Settings** เพื่อบันทึกการตั้งค่า
5. คลิกปุ่ม **🔑 Sign In with Google** เพื่อลงชื่อเข้าใช้และอนุญาตสิทธิ์การเขียนไฟล์ใน Google Drive

---

## 🎬 วิธีใช้งานในหน้า BigSeller (สแกนตรวจสินค้า)

1. เปิดหน้าเว็บตรวจสินค้าของ BigSeller: 
   👉 [https://www.bigseller.com/web/order/scanToInspection.htm](https://www.bigseller.com/web/order/scanToInspection.htm)
2. เมื่อหน้าเว็บโหลดเสร็จ คุณจะเห็น **กล่องหน้าจอกล้องลอยน้ำ (Floating Camera Panel)** ปรากฏขึ้นที่มุมขวาบนของหน้าเว็บ BigSeller
   * สามารถกด ย่อ/ขยาย หน้าจอกล้องได้
   * สามารถใช้เมาส์ลาก (Drag) หัวข้อกล่องเพื่อย้ายตำแหน่งไปวางจุดอื่นได้ตามต้องการ
   * สามารถเลือกกล้องที่ใช้งานได้ผ่านทางไอคอนฟันเฟือง ⚙️ ในกล่องกล้องลอยน้ำ
3. **ขั้นตอนการอัดอัตโนมัติ (Auto-Record)**:
   * **เริ่มอัด**: เมื่อพนักงานโฟกัสที่ช่องป้อนข้อมูลและทำการสแกนบาร์โค้ดพัสดุ (หรือกด Enter ในช่องสแกน) ระบบจะตรวจสอบเลขพัสดุจาก `<span class="tackingNo">` และสั่งเริ่มอัดวิดีโอทันที โดยในวิดีโอจะฝังลายน้ำสีแดงรอบกรอบ, หมายเลขแทร็คกิ้ง, ชื่อขนส่ง, และเวลาปัจจุบันแบบเรียลไทม์
   * **หยุดอัดและอัปโหลด**: เมื่อตรวจสอบสินค้าเสร็จ พนักงานกดปุ่ม **"ล้างข้อมูล & สแกนถัดไป"** บนหน้าเว็บ BigSeller ระบบจะจับเหตุการณ์การกดปุ่มนี้ สั่งหยุดอัดวิดีโอ และอัปโหลดไฟล์ขึ้น Google Drive อัตโนมัติในพื้นหลังทันที
   * **กรณีสแกนต่อเนื่อง**: หากพนักงานสแกนกล่องถัดไปโดยไม่ได้กดล้างข้อมูล ระบบจะหยุดอัดและอัปโหลดไฟล์กล่องเดิม จากนั้นเริ่มบันทึกวิดีโอกล่องใหม่ให้ทันทีโดยอัตโนมัติ
4. **การดูรายงานย้อนหลัง**:
   * คลิกที่ไอคอนส่วนเสริม 📦 เพื่อดูจำนวนกล่องที่แพ็คเสร็จในวันนั้นๆ
   * คุณจะเห็นรายการพัสดุทั้งหมดที่มีการบันทึก พร้อมลิงก์ **"↗ เปิดวิดีโอบน Drive"** สำหรับการตรวจสอบความถูกต้องย้อนหลังได้อย่างรวดเร็ว

---

### ⚠️ หมายเหตุการใช้งานกล้อง
* เมื่อใช้งานครั้งแรก Chrome จะแจ้งเตือนขอสิทธิ์เข้าใช้งานกล้องเว็บแคมบนโดเมน `bigseller.com` กรุณากด **"อนุญาต" (Allow)** เพื่อให้ระบบทำงานได้เต็มประสิทธิภาพ
